# 🔗 Linktree Clone – Aramizkingh

A modern and responsive Link-in-bio web application inspired by Linktree. Built to showcase personal brand, social links, and projects in a clean and interactive UI.


## 🚀 Live Demo
👉 https://aramizkingh.github.io/project-2-linktree/


## ✨ Features

- 🌗 Dark / Light mode toggle  
- 🎨 Beautiful gradient UI design  
- 📱 Fully responsive (mobile & desktop)  
- 🔗 Social media links (GitHub, LinkedIn, Instagram, YouTube)  
- ⚡ Smooth hover & click effects  
- 🧠 JavaScript interactivity (DOM manipulation)
